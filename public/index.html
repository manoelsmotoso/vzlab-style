Quick start

Copy-paste the stylesheet <link> into your <head> before all other stylesheets to load our CSS.

<link rel="stylesheet" type="text/css" href="https://unpkg.com/@vzlab/style@2.5.3/dist/main.css">

You need to import the fonts too:

<link rel="stylesheet" type="text/css" href="https://fonts.googleapis.com/css?family=Titillium+Web:400,600">

We are using Ionicons, so if you want to use the same icons copy-paste <link> into your <head>:

<link rel="stylesheet" type="text/css" href="https://unpkg.com/ionicons@4.5.0/dist/css/ionicons.min.css">

And if you want to use a Grid system you can use the following:

<link rel="stylesheet" type="text/css" href="https://unpkg.com/flexboxgrid@6.3.1/dist/flexboxgrid.min.css">
